About

Razorpay enables secure online transactions for eco stay bookings, tribal marketplace sales, and government tourism payments. It supports UPI, credit/debit cards, wallets, and net banking — making it the ideal payment partner for EcoKumari's diverse user base across rural and urban India.

Features

  • Secure payment gateway with 256-bit SSL
  • Automated settlement to host bank accounts
  • Tribal artisan payout tracking & reconciliation
  • Refund management for cancelled bookings
  • GST compliance and automated invoice generation
  • Webhook support for real-time payment events

Setup Instructions

  1. 1
    Create Razorpay Account

    Visit razorpay.com and register with your business details and PAN number.

  2. 2
    Generate API Keys

    Navigate to Settings → API Keys in your Razorpay dashboard and generate a key pair.

  3. 3
    Paste Keys in EcoKumari

    Go to Settings → Integrations → Razorpay and enter your API Key ID and secret.

  4. 4
    Enable Webhook

    Configure the webhook URL https://api.ecokumari.in/webhooks/razorpay in your Razorpay dashboard.

Requirements

  • Active EcoKumari workspace
  • Admin role or higher
  • Verified business account on Razorpay
