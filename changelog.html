v2.4.0 Major Release
January 15, 2025

Tourism Dashboard 2.0 & Advanced Impact Analytics

This release introduces a redesigned dashboard experience and powerful analytics tools to help monitor tourism growth and tribal livelihood impact across all registered destinations.

✨ New Features

  • Fully redesigned Tourism Admin Dashboard with customizable widget layout
  • Widgets for bookings, artisan sales, eco impact score, and compliance status
  • Advanced analytics with custom date range filters and CSV export
  • Lake visitor heatmap visualization for destination management
  • Dark mode support across all dashboard screens

🚀 Improvements

  • 50% faster page load time across dashboard and reports
  • Improved search with location-based and artisan-specific filters
  • Enhanced mobile booking experience with swipe gestures
  • Simplified host approval workflow — reduced to 3 steps

🐛 Bug Fixes

  • Fixed delayed booking confirmation notifications in WhatsApp integration
  • Resolved tribal payout calculation mismatch for artisans with multiple listings
  • Corrected map pin duplication issue for co-located eco stays

⚠️ Breaking Changes

  • Legacy reporting endpoints deprecated — please migrate to v2 APIs before March 30, 2025

v2.3.0 Minor Update
December 20, 2024

Tribal Marketplace Expansion

Expanded support for artisan onboarding and marketplace growth, enabling communities to list, manage, and sell their products with greater ease.

✨ New Features

  • Bulk product upload for artisans (up to 100 products via CSV)
  • Multi-language product descriptions — Tamil, Hindi, English
  • Revenue analytics per artisan with monthly trend charts
  • UPI payout tracking dashboard for real-time settlement status

🚀 Improvements

  • Improved vendor verification process — down from 7 to 3 days
  • Faster admin approval system with batch processing support
  • Simplified GST input format with auto-validation

🐛 Bug Fixes

  • Fixed payout status update delay in artisan dashboards
  • Resolved broken product image previews in marketplace listings
v2.2.0 Major Release
November 30, 2024

Eco Stay Booking System Launch

First full release of the eco-tourism booking engine, enabling travelers to discover, book, and pay for verified eco stays across Kanniyakumari and surrounding regions.

✨ New Features

  • Verified eco stay listings with 75-point sustainability audit badge
  • Real-time availability calendar with instant booking
  • Secure payment gateway integration (Razorpay UPI & Cards)
  • Booking confirmation via SMS and email with QR check-in code

🚀 Improvements

  • Improved listing filters — price range, eco rating, amenities, and proximity
  • Enhanced mobile UI with a streamlined 3-step booking flow

🐛 Bug Fixes

  • Fixed duplicate booking prevention when sessions overlap
  • Corrected refund calculation logic for partial-night cancellations
